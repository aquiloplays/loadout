# DEPRECATED

This PyInstaller tray companion has been superseded by the browser extension at
`Loadout/extension-kindle/` (release `extension-kindle-v0.1.0`).

Why the switch: the extension runs inside your already-authenticated
`read.amazon.com/notebook` tab, so there is no Selenium driver, no cookie
capture, no DPAPI, no PyInstaller, and no autostart. Same ingest contract, same
Knowledge Vault.

The `companion-kindle-v0.1.0` exe release is kept for history but marked
superseded. Use the extension instead.
