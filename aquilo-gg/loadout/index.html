What's in the kit

⚡ Bolts wallet

Cross-platform points wallet with sub multipliers, Patreon multipliers, and a daily-streak bonus. Earn from chat, subs, gifts, raids, TikTok coins, CC coins, channel points. Spend with !gift and !boltrain. One unified OBS overlay handles leaderboard, earn toasts, bolt rain, streak banners, and gift bursts.

👑 Apex top-viewer

One viewer holds the spot. Every spend event from anyone else chips away at their HP. Cross-platform damage — TikTok gifts hit just like Twitch subs. Finisher takes the crown. Reign timer, animated HP bar, dethrone splash.

🎯 Daily Check-In

Twitch channel-point reward or !checkin on any platform. Overlay shows the viewer's avatar, sub-tier flair, Patreon flair, and rotating stream stats underneath.

🤖 AI shoutouts

On raid, the bot pulls the raider's last category and writes a hype, specific shoutout via Anthropic Claude or OpenAI. Bring your own key, or use the bundled key — a thank-you perk for Tier 3 patrons.

🛡️ Hate-raid detector

Pattern-based detection: short window, brand-new accounts, similar messages. DMs the broadcaster, no chat noise.

📊 Counters & goals

!deaths +1, !wins +1, custom counters. Follower / sub / bit / coin goals with kind-themed progress bars. Live OBS overlays update on every change.

🚂 Sub raid trains

Burst detection at 3, 6, 10, 20 subs in 60s. Plus synthetic TikTok hype train for TikFinity gifts (TikTok has none natively).

🪙 Crowd Control coins

Per-viewer leaderboard for CC effect spend. !cccoins top 5 this stream, !cccoinsall all-time, !mycoins personal stats.

📨 Discord live status

Auto-posts a go-live embed. Edits in place when you change title or category. Strikethroughs and footers it when you go offline. End-of-stream recap to a separate webhook.

🔌 Webhook inbox

HTTP listener for Ko-fi, Throne, Patreon, custom apps. Secret-authenticated. Publishes to the Aquilo Bus AND can fire any SB action by id.

👥 VIP rotation

Weekly auto-rotation: promotes most engaged non-VIPs, demotes least active VIPs. Mods can force !viprotate any time. Posts to Discord.

🤫 Chat noise control

Global cap of 30 messages/minute total. Per-area mute. Per-command cooldowns. Quiet mode silences ambient chat while overlays keep updating. !loadout quiet.

5 minute install

  1. Download Loadout.dll and loadout-import.sb.txt from the latest release.
  2. Drop the DLL into <Streamerbot>/data/Loadout/. (Optional — the bootstrap action will download it for you on first run.)
  3. Import the bundle. In Streamer.bot click Import top-right and paste the contents of loadout-import.sb.txt.
  4. Run Loadout: Boot. Right-click in the SB action panel → Run Now. Or restart SB.
  5. Walk the wizard. Pick what you want enabled. Click Recommended if you don't want to think about it.
